# Nekoplayer

基于vue构建的第三方网易云播放器和本地播放器

[😺查看Release](https://github.com/kiyonya/Nekoplayer/releases)
[📑反馈](https://github.com/kiyonya/Nekoplayer/issues)

## 应用特点
- 支持网易云账号登录，查看个人歌单，收藏
- 支持浏览网易云歌单，专辑，艺术家等
- 支持网易云的个性推荐和私人FM
- 支持获取网易云音频，歌词，封面等信息
- 应用主题自定义
- 播放页UI和弹簧歌词滚动
