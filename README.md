# Nekoplayer
基于vue和electron构建的第三方网易云播放器和本地播放器应用
**😺希望可以带给你更好的使用体验~**
## 应用特点
- 支持账号登录，查看个人歌单，收藏
- 支持浏览歌单，专辑，艺术家等
- 支持个性推荐和私人FM
- 支持获取和匹配音频，歌词，封面等信息
- 支持一起听
- 支持本地音乐管理，元数据查看，歌词封面匹配，本地音乐集创建（开发中，基本完成）
- 应用主题自定义
- 现代化UI以及更好的歌词动效
